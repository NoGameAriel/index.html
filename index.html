What's happening

These aren't isolated frustrations. They're structural patterns — and they have names, sources, and points where repair becomes possible.

You've noticed —
The same problems return even after something was "fixed."

New leadership, new policy — the pattern reasserts itself within months.

You've noticed —
People go through the motions. Meetings happen. Nothing changes.

Participation on the surface, silence underneath. Real problems stay unspoken.

You've noticed —
It's not that anyone is evil. The structure pushes toward bad outcomes.

You can't point to a villain. The incentives and feedback loops make failure almost inevitable.

Common signal collapse patterns
False Resonance

The system looks like it's responding. Announcements are made, committees formed, language changes. The underlying structure doesn't move.

→ Legitimacy capacity degraded
Suppressed Signal

Accurate information exists inside the system but stops flowing to where decisions are made. The people with power to act are the last to know.

→ Reality capacity degraded
Defensive Messaging

The system has stopped learning. Every failure is absorbed into narrative rather than correction. Repeated errors get different names, not different responses.

→ Adaptive capacity degraded
Polarized Identity Formation

When coordination breaks down, group membership becomes the primary currency. Analysis is replaced by allegiance. The problem becomes tribal rather than structural.

→ Coordination + normative capacity degraded
How to read it
Integrity is not moral perfection. It is structural alignment between signal, decision, and outcome. When that alignment breaks, distortion emerges. When it is restored, legitimacy stabilizes.
Five core capacities

Long-term stability requires functionality across all five. Breakdown in any one produces distortion at the experiential level.

01
Reality Capacity
Can the system see clearly?

When this breaks: the system acts on distorted inputs, ignores accurate feedback, punishes honest reporting.

Restored by: structured feedback channels, transparent signal logs.
02
Legitimacy Capacity
Does anyone still trust it?

When this breaks: compliance replaces participation, performance replaces engagement.

Restored by: transparent decision matrices, consistent follow-through.
03
Coordination Capacity
Can people actually work together?

When this breaks: blame cycles form, silos harden, parallel efforts cancel each other.

Restored by: cross-functional review cadences, shared accountability.
04
Adaptive Capacity
Can it learn from mistakes?

When this breaks: same failures repeat, defensive messaging replaces correction.

Restored by: protected error-reporting, binding acknowledgment of failure patterns.
05
Normative Orientation
Does it know what it's for?

When this breaks: mission drifts, group identity replaces shared purpose, polarization sets in.

Restored by: explicit shared goals that exist independent of current leadership.

In practice

The five capacities aren't abstract. When each one fails, it produces observable, nameable conditions — and each points toward a specific repair lever. Here's what that looks like mapped onto a city council under stress.

Capacity Observed condition Distortion signal Repair lever
Reality Public comment systematically ignored Signal suppression Structured response logs with binding acknowledgment
Legitimacy Declining voter turnout across three cycles Performance politics Transparent decision matrix published before votes
