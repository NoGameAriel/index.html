02 — Recognition

These patterns
have names.

What you're experiencing at the surface is a distortion — a symptom of something structural happening underneath. A sample of common signal collapse patterns:

False Resonance

The system looks like it's responding. Announcements are made, committees are formed, language changes. The underlying structure doesn't move. This is the most common form of institutional stall.

→ Legitimacy capacity degraded
Suppressed Signal

Accurate information exists inside the system but stops flowing to where decisions are made. The people with power to act are the last to know what's actually happening.

→ Reality capacity degraded
Defensive Messaging

The system has stopped learning. Every failure is absorbed into narrative rather than correction. Repeated errors get different names, not different responses. Reform language without reform behavior.

→ Adaptive capacity degraded
Polarized Identity Formation

When coordination breaks down and shared purpose collapses, group membership becomes the primary currency. Analysis is replaced by allegiance. The problem becomes tribal rather than structural.

→ Coordination + normative capacity degraded
These four represent core archetypes. The full distortion lexicon is in the corpus.
03 — The Lens

Five capacities that explain
what's actually breaking

Each distortion above traces back to one of five structural capacities. Knowing which one is failing tells you where repair is actually possible — and where it isn't.

Integrity is not moral perfection. It is structural alignment between signal, decision, and outcome. When it degrades, distortion emerges. When it's restored, the system can function again.

CAP — 01 ◎
Reality Capacity
Can the system see clearly?

When this breaks: the system acts on distorted inputs, ignores accurate feedback, punishes honest reporting.

Restored by: structured feedback channels, transparent signal logs, error acknowledgment obligations.
CAP — 02 ◈
Legitimacy Capacity
Does anyone still trust it?

When this breaks: compliance replaces participation, performance replaces engagement, people exit or perform.

Restored by: transparent decision matrices, published rationale, consistent follow-through.
CAP — 03 ⬡
Coordination Capacity
Can people actually work together?

When this breaks: blame cycles form, silos harden, parallel efforts cancel each other out.

Restored by: cross-functional review cadences, shared accountability structures.
CAP — 04 ↻
Adaptive Capacity
Can it learn from mistakes?

When this breaks: same failures repeat, defensive messaging replaces correction, the system hardens.

Restored by: protected error-reporting, binding acknowledgment of failure patterns.
CAP — 05 ◇
Normative Orientation
Does it know what it's for?

When this breaks: mission drifts, group identity replaces shared purpose, polarization sets in.

Restored by: explicit, shared goals that exist independent of current leadership or cycle.

Why the instruments work consistently

Built on Condition Preservation Theory

Every instrument in Field & Signal produces consistent results across contexts — workplaces, governments, communities — because they all share the same structural foundation.

That foundation is CPT: a constraint engine that runs before any intervention is designed. It answers one question first.

Is this system in a state where change can actually land?
If not, even a well-designed, well-resourced intervention will be absorbed, inverted, or destroyed by the conditions it enters. CPT identifies whether those conditions are present before you spend effort designing a response.
