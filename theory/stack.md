# The Five-Level Stack

| Level | Name | Function |
|---|---|---|
| 1 | Domain | Defines what is real and what counts as change |
| 2 | Feedback | Reinforces behavior within that definition |
| 3 | Drift | Accumulated divergence from external reality |
| 4 | Meta-Feedback | Detects drift independent of reinforcement incentives |
| 5 | Field & Signal | Human operational layer performing meta-feedback in lived systems |
